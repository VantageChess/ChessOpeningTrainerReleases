# ♟️ Chess Opening Driller

**Master your chess openings through focused, repetitive practice.**

Chess Opening Driller is a desktop application that helps you memorize and internalize chess openings by drilling them move by move. Load any opening repertoire, practice as White or Black, and build the muscle memory that makes your preparation rock-solid.

---

## 📥 Download

Head to the [**Releases page**](../../releases) to download the latest version for your platform:

- 🍎 **macOS** — `.dmg` installer
- 🐧 **Linux** — `.deb` package or `.AppImage`

---

## ✨ Features

- **Opening drill mode** — practice your repertoire move by move with instant feedback
- **Multi-language support** — available in English, Italian, Spanish, and French
- **Lightweight & fast** — no internet connection required, everything runs locally
- **Simple to set up** — load a PGN file and start drilling in seconds
