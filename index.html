What it does

A study tool built around repetition, recall, and real repertoire work

Chess Opening Trainer is a desktop app for drilling chess opening repertoires in a focused, practical way. Instead of passively reading moves, you actively recall them and reinforce the lines you want to keep ready for real games.

Features

Built for players who want structured opening practice

01

Interactive drilling

You can import opening data, organize your material into multiple libraries, and run training sessions that guide you through variations one decision at a time.

02

Multiple opening libraries

Keep separate repertoires, test collections, or study sets without forcing everything into one tree.

03

PGN import

Bring in opening data from existing sources so your study material starts from your real repertoire.

04

Board themes and preferences

Adjust the training environment with board themes, piece sets, and interface preferences that suit your study habits.

05

Multilingual interface

The app and this site are designed to support multiple languages for a more comfortable learning experience.

Installation

Install the build that matches your platform

macOS

  1. Download the current `.dmg` release build.
  2. Open the disk image and move the app into your Applications folder.
  3. If macOS warns that the app is from an unidentified developer, open System Settings > Privacy & Security and choose “Open Anyway”.

Linux .deb

  1. Download the `.deb` package.
  2. Install it with your system package manager or package installer.
  3. This package supports in-app updating on Linux.

Linux .AppImage

  1. Download the `.AppImage` file.
  2. Mark it as executable if your system does not do that automatically.
  3. Behavior can differ from the `.deb` package depending on your distribution and desktop setup.

FAQ

Practical questions

Is the app free to use?

The public release process supports free distribution, but you should check the current release notes or project messaging for the most up-to-date licensing and pricing details.

How do I update it?

Linux users can rely on in-app updating when supported by their package. macOS users should download the latest `.dmg` from GitHub Releases until signed updates are available.

Why does macOS warn me before opening it?

Current macOS builds are not yet signed and notarized, so Gatekeeper may show a warning. You can allow the app from Privacy & Security if you trust the download source.

Which Linux package should I choose: `.deb` or `.AppImage`?

Choose the `.deb` package on Debian-based systems when you want tighter package-manager integration. Pick `.AppImage` if you prefer a portable single-file build or your setup does not use `.deb`.

Does it work offline?

The training workflow is designed for desktop use and core study features can work offline once the app is installed, though update checks obviously require internet access.

Will Windows support be added?

Windows support is planned, but there is no public release yet. The website includes a placeholder so it can be added cleanly later.
